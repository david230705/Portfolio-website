# Personal Portfolio Website

A responsive personal portfolio website built to showcase my development work, technical skills, and projects through a modern single-page interface.

## About the Project

This project was developed from scratch using HTML, CSS, and JavaScript with a focus on creating a clean, interactive, and responsive user experience.

The design includes custom sections, animations, and reusable styling components to create a modern developer portfolio.

## Built With

- HTML5
- CSS3
- JavaScript
- Three.js
- Google Fonts

## Features

- Responsive layout for different screen sizes
- Interactive UI elements
- Custom animations and effects
- Project showcase sections
- Embedded document access
- Modern dark-themed design
## Deployment

The website is deployed using Vercel.
